Deployment:

Language used:
   
1.Html

2.CSS

3.Javascript

4.Python

Procedure:

1.Front end of the app is designed using HTML,CSS and JavaScript.

2.Backend end of the app is implemented using Python.

How the app works:

User types a question and click on the submit button a POST request(https request) is sent from JavaScript to Python.Python sends the response in the form to JSON to JavaScript.The result is displayed in the form of similar questions on the webpage.
