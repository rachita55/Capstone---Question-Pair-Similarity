Technique used:

1. Removing tags

2. Expanding english language

3. Regex Function

Step 2: Find and remove tags like closed and duplicate at the end or middle of the sentence.

Step 3: Create a dictionary and use it for expanding english language.

Step 4: Use regex function to remove symbols like (),[],-,smileys and images.
